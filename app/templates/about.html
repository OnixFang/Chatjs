v0.1A

Current features

  • User account:
    • Register: Create a new user with username, password, first name, last name and a pass code to recover your password.
    • Login: Log in with your username and password.
    • Reset passowrd: Reset your password with your username and pass code.
  • User menu:
    Open menu to log out, see the about page and other features to come.
  • User search bar:
    Search for an user to chat with by username, first name or last name, then click the desired contact.
  • Chat page:
    Send messages real time to other users. You can press the Enter key to send the message.
